Sass Switch Mixin.

Create switches width one line of css!

========== HTML ==========

The Html is composed by :

  • A container. It can be any element you want, div, p, a, span
  • A checkbox input with data-type=switch
  • The label

Here is a full example. 

	
		 <p class="switch ios rounded">
			 <input type="checkbox" name="my-switch" data-type="switch">
			 <label for="my-switch"> <label>
		 <p>

The use of classes depends on what type of css implementation is done, sass or css.

You can use the switch with Sass or as standalone css file.

========== CSS ==========

Use without Sass.

  • You need to import the file switch.css or switch.min.css
  • There are 2 ways of importing it :

  • Via @import. Be aware that this will be a new http request
  • Copy and paste the styles in you main css file, possibly at the end of the file.(recommended)

Once you have imported the css file you can create a switch just adding classes to your HTML code(For the HTML Code refer to the html part).

Possible classes:

  • switch -> Creates a basic switch button in ios style.
  • ios -> Ios Button
  • rounded -> Used together with ios, Rounded Button
  • square -> Used together with ios, Square Button
  • switch material -> Material Design style Button

========== SASS ==========

To use the mixin with sass.

  • import the mixin (_switch.scss) in your sass file
  • @import "switch";
  • include the mixin, possibly at the top of the declaration:
.switch { @include switch(); }

Parameters :

  • $type - String - It can be either "ios" or "material". Default value is "ios".
  • $width: Number - Any number (with no unit) to define the width. Default value is 50
  • $subtype: String - Must be declared if the type is 'ios'. Can be 'rounded' or 'square'. Default
  • value null or 'rounded'.
  • $materialBackground - Hex Value - Must be declared if type is 'material'. Defines the background
  • color. Default value: #8BC34A.
  • $toggleBackground - HHex value - Defines the background for the toggle button. Default value is #ffffff.

If no parameters are defined these are the default values:

  • $type: 'ios'
  • $width: 50
  • $subtype: 'rounded'
  • $materialBackground: #8BC34A
  • $toggleBackground: #ffffff

Full Example: 

.my-switch {
	@include switch(
	$type: 'material',
	$width: 60, 
	$materialBackground: #FFEB3B,
	$toggleBackground: #f9f9f9
	);
}		

<p class="my-switch">
	<input type="checkbox" name="my-switch" data-type="switch">
	<label for="my-switch"> <label>
<p>
